# Game Server
## Description
Basic server project to the game. This server use NestJS with Typescript, Fastify instead of Express, Prisma ORM (with Migration) to connect in a SQLite database (for a while) and Swagger already configured.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
